Fonts & UI Components

Design | Font hierarchy | UI Cards

A mockup landing page using fonts and grey scales only. The main tasks were to create a font hierarchy with Type Scale and to design UI components using CSS flex and observed design conventions for cards.

SEM3 | Assignment | 2 days

Process

After gathering some inspiration from other landing pages, I decided I wanted to work with a capitalized font as the display and header font. I used Type Scale to find a good balance for the font sizes, and went with the "Augmented Fourth" scale. I then adjusted line height, margins and font weights accordingly for the headings and paragraphs.

I kept the UI cards very simple, focusing on the layout and presentation of the different elements inside. I used the design conventions for cards that we observed in class, such as very light shadows and only slight corner rounding.
